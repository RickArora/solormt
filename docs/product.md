# SoloRMT Product Notes

## MVP

SoloRMT helps independent RMTs manage clients, appointments, SOAP notes, payments, and clinic metrics from one dashboard.

## Primary User

- Ontario RMTs
- Canadian RMTs
- Solo practitioners
- Home clinic owners
- Mobile massage therapists

## Success Criteria

An RMT can create an account, add a client, book an appointment, create a SOAP note, accept payment through Stripe, and view metrics without touching the database.
